自動設定されるCSS変数
/* :root に自動設定 */
--windowHeight  /* window.innerHeight */

--footerHeight  /* footer 高さ */
--scrollTop     /* 現在スクロール量 */
--scrollMax     /* 最大スクロール量 */

/* body クラス（スクロール量に応じて自動付与） */
.is-scrollStart    /* scrollTop > 10 */
.is-scrollOver100  /* scrollTop > 100 */
.is-scrollOver200  /* scrollTop > 200 */
.is-scrollOver400  /* scrollTop > 400 */
.is-scrollEnd      /* 末尾到達 */

.hero { min-height: var(--svh); }
.l-header { height: var(--headerHeight); }
/* ヘッダー: スクロールが始まったら背景付与 */

LoadSequence LoadSequence

.l-loading があるとスクロールロックして lazybackground:complete イベントを待機。完了後にフェードアウトして Lenis を起動。

HTML + CSS
<!-- head 内: body に .is-ready を付与しておく -->
<body class="is-ready">

<!-- ローディング画面（オプション） -->
<div class="l-loading">Loading...</div>

/* CSS */
.l-loading { position:fixed; inset:0; opacity:1; transition: opacity 0.5s; }
body.is-loadingEnd .l-loading { opacity:0; pointer-events:none; }
bodyクラスの変化フロー
is-ready        ← HTML初期状態
↓ DOMContentLoaded
is-loadingStart ← ローディングアニメ開始
↓ lazybackground:complete（.is-preload 画像の読み込み完了）
is-loadingEnd   ← ローディングフェードアウト
↓ 600ms後
is-load         ← Lenis起動 / コンテンツアニメ開始

ParallaxImages ParallaxImages

.js-sa__image に --thisProgressBottom を毎フレーム更新。CSS transform でパララックスを実現。

HTML + CSS
<!-- 親でクリッピング -->
<div class="image-wrap">
  <div class="js-sa__image"></div>
</div>

.image-wrap { position:relative; overflow:hidden; height:300px; }
.js-sa__image {
  position: absolute;
  inset: -60px 0; /* パララックス分を上下にはみ出す */
  background-image: url("photo.webp");
  background-size: cover;
  /* 0.5 = 中央値。±0.5 の範囲でtransform */
  transform: translateY(calc((var(--thisProgressBottom) - 0.5) * -80px));
}

Accordion デモ
Q. .js-accordion の基本構造は？
.js-accordion > .js-accordion__head + .js-accordion__body > .js-accordion__body-inner の3層構造が必要です。
Q. 初期展開するには？
head に .is-active クラスを付与しておくと初期展開状態になります。
Q. 複数アコーディオンを排他制御するには？
index.js のデフォルトは非排他です。排他制御が必要な場合はカスタマイズしてください。

BrWrap BrWrap

.p-text / .p-lead 内で可視 <br> が2つ以上連続する箇所を段落区切りとし、前後のテキストを span.is-brwrap で囲む。

HTML → 処理後
<!-- 処理前 -->
<div class="p-text">
  第1段落<br><br>
  第2段落<br><br>
  第3段落
</div>

<!-- 処理後（自動変換） -->
<div class="p-text">
  <span class="is-brwrap">第1段落</span>
  <br class="is-hidden"><br class="is-hidden">
  <span class="is-brwrap">第2段落</span>
  ...
</div>

/* CSS: .is-brwrap を block にして段落扱い */
.p-text .is-brwrap { display: block; margin-bottom: 0.75em; }
.p-text br.is-hidden { display: none; }
処理後: 連続brが段落に変換される
第1段落テキストです。改行は br タグで表現します。

第2段落テキストです。連続した br が2つ以上で段落区切りと判定されます。

第3段落テキストです。.is-brwrap が display:block で各段落を区切ります。
